Замещаемые ссылки - это множество всех переменных sc-узлов, таких, что каждый из них удовлетворяет следующим условиям:

  • замещаемая ссылка принадлежит как минимум одному шаблону поиска книг;
  • замещаемая ссылка является трансляцией ровно одной sc-ссылки, значение которой должно быть подставлено на место замещаемой ссылки во время поиска книг по шаблону.
